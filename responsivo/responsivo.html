@Media Queries

CSS uses the @media rule to detect info about the device being used to access the website. For example, it can be used to detect the size of the device, its capabilities, and its orientation. for this course, we will be using it to detect the device size, but keep in mind that it is capable of much more.

Desktop width greater than or equal to 1024

Tablet width between 481 and 1023

Movil width less than or equal to 480px
